# 🌷 Tulip Garden

A cute interactive tulip garden — water the flowers and watch them grow!

## Features
- 9 colorful tulips that grow when watered
- Animated sun, drifting clouds, and swaying stems
- Water drop effect on button click
- Reset and regrow as many times as you like

## How to use
Just open `index.html` in your browser — no build step needed.
